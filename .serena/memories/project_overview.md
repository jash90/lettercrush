# Project Overview: LetterCrush (WordGrid)

## Purpose
LetterCrush (internally "WordGrid") is a mobile word puzzle game that combines Match-3 mechanics with word creation. It aims to be an engaging, casual/mid-core game for iOS and Android.

## Genre
Word Puzzle / Match-3 Hybrid.
"Candy Crush meets Scrabble".

## Key Features
- **Hybrid Gameplay:** Swap letters on a 6x6 grid to form words.
- **Modes:** Classic (2 min), Campaign (Puzzle), Daily Challenge, Word Hunt, PvP (planned), Zen.
- **Progression:** Player levels, Personal Dictionary, Achievements, Battle Pass.
- **Monetization:** Hybrid (IAP + Ads + Subscription).

## Status
- **Version:** 1.0 (MVP/Development)
- **Languages:** English, Polish (Simplified v1.0, Full v2.0).

## Platforms
- iOS
- Android
- Web (via Expo, likely for dev/testing)
