## 2. Advertising Strategy (v1.0)

### 2.1 Interstitial Ads - Implemented

| Parameter | Value |
|-----------|-------|
| **Placement** | After "Play Again" button tap |
| **Trigger** | Before starting new game |
| **Frequency** | Every new game |
| **Skip Option** | After 5 seconds |
| **Network** | Google AdMob |

### 2.5 User Experience Considerations

| Principle | Implementation |
|-----------|----------------|
| Non-blocking | Ad load failures don't prevent gameplay |
| Predictable | Ads only on "Play Again", never during game |
| Respectful | No mid-game interruptions |
| Performant | Preload ads for instant display |
